   - index.html

part 2


Extend the existing "Student Toolkit Dashboard" website by adding JavaScript functionality.

The project uses HTML, CSS, and JavaScript and already includes:
- A Navigation Bar
- A Sidebar with "Upcoming Events"
- A Main Dashboard area

Add the following interactive tools:

1. GPA Calculator

Create a GPA Calculator section inside the dashboard.

Requirements:
- Allow the user to enter multiple courses.
- Each course should include:
    • Course Credits (number input)
    • Grade (dropdown or text input such as A, B, C, D, F)
- Add a button labeled "Calculate GPA".

Functionality:
- When the button is clicked:
    • Convert grades to grade points (A=4, B=3, C=2, D=1, F=0).
    • Multiply grade points by course credits.
    • Calculate the total GPA using the formula:

      GPA = (Sum of (grade points × credits)) / (Total credits)

- Display the calculated GPA on the page.

2. Pomodoro Study Timer

Create a Pomodoro Timer section inside the dashboard.

Requirements:
- Display a 25-minute countdown timer (25:00).
- Include:
    • Start Button
    • Reset Button
- The timer should count down every second.

Functionality:
- When the Start button is clicked:
    • The timer begins counting down.
- When the Reset button is clicked:
    • The timer resets back to 25:00.
- When the timer reaches 00:00:
    • Play a bell sound OR show a browser alert saying "Time's up! Take a break."

Technical Requirements:
- Use JavaScript for all logic.
- Use DOM manipulation to update the timer and GPA result.
- Organize the code inside a separate file called script.js.
- Keep the interface clean and easy to use.

Output:
Provide the complete code for:
- index.html
- style.css
- script.js
